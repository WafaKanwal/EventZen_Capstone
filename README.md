EventZen is a comprehensive event management platform designed to provide a seamless experience to organize and manage events. The project features a React.js based website for a responsive and interactive user interface, along with three separate backend services: Event Booking Service (Spring Boot) for managing events and bookings, User Service (Spring Boot) for user authentication and management, and Venue Service (Node.js) for handling venue and vendor information. EventZen allows users to register and log in securely, browse and search for events, make bookings, and manage their participation, while admins can oversee events and operations efficiently. The platform uses JWT-based authentication for security and integrates databases for storing user, event, and venue data. This modular architecture ensures a scalable, maintainable, and fully functional event management solution.
